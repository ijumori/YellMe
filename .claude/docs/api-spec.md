# Claude API フィードバック仕様

## モード別プロンプト方針

| モード | 方針 | 禁止事項 |
|--------|------|---------|
| `.praise` | 3つ以上の具体的褒めポイントを文脈から発見 | 抽象的なお世辞 |
| `.empathy` | 評価・アドバイスなし。ただ寄り添う | 「でも」「しかし」 |
| `.advice` | 褒め→1つだけ小さなアドバイス | 押しつけ・複数提案 |
| `.courage` | 一歩踏み出せる背中を押す言葉 | 比較・プレッシャー |

## 共通禁止事項
- 批判・否定・比較
- 「でも」「しかし」「もっと〜すべき」
- 具体的文脈なしの抽象的フィードバック
